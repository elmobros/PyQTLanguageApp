ThaiForBeginners is a PyQt5 Python Application
Author: Sean Nomoto

To run application, you must run the file 'MainWindow.py' with Python 3.  

This application is based on the Thai For Beginners Book by Benjawan Poomsan Becker, and is not
intended to infringe copyright.  This is only a sample application created as a proof
of concept and instructional exercise in creating a GUI-based PyQt5 Application.
